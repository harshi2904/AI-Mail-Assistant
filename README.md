# 🤖 AI Mail Insight Assistant  
### Built with Machine Learning + Generative AI (SymphonyAI-Style Project)

An **enterprise-grade AI application** that classifies, summarizes, and auto-replies to business emails using a hybrid **Machine Learning + Generative AI** pipeline.  
Inspired by SymphonyAI’s *AI Copilot for Work*, this project demonstrates how GenAI can automate email triage and improve enterprise communication efficiency.

---

## 🌟 Features

- 📂 **Email Intent Classification** — Automatically detects the category (HR, IT Support, Finance, Admin, or General).  
- 🧠 **Generative AI Summarization** — Extracts the key intent from each email using the BART transformer model.  
- 💌 **AI-Powered Replies** — Generates short, polite, and context-aware professional responses.  
- 🧮 **Hybrid AI Stack** — Combines TF-IDF + Logistic Regression (ML) with Hugging Face Transformers (GenAI).  
- 🌐 **Interactive Streamlit App** — Real-time analysis with instant results.  
- 📊 **Result Logging** — Saves every analyzed email, summary, and reply to CSV for future insights.
